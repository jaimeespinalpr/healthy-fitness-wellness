# Healthy Fitness and Wellness

Static English website for Fiat Fitness Project. It includes original site imagery, optimized WebP assets, internal program pages, a premium program directory, organized workout studio filters, SEO metadata, sitemap, mobile navigation, testimonials, and an official-contact handoff.

## GitHub Pages

This project is published from the `main` branch using GitHub Pages from the repository root.

## Files

- `index.html`: main site structure.
- `styles.css`: visual and responsive styles.
- `script.js`: interactive weekly-goal selector and contact-form confirmation.
- `assets/`: Fiat Fitness Project images used by the published page.
- `programs/`: internal program overview pages.
- `sitemap.xml` and `robots.txt`: basic SEO discovery files.
