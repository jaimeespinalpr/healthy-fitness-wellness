# Healthy Fitness and Wellness

Static English website for Fiat Fitness Project. It includes the original site imagery, a centered brand masthead, program options, a full directory of public program/course/workout/account links, a weekly focus selector, welcome section, testimonials, and contact form.

## GitHub Pages

This project is published from the `main` branch using GitHub Pages from the repository root.

## Files

- `index.html`: main site structure.
- `styles.css`: visual and responsive styles.
- `script.js`: interactive weekly-goal selector and contact-form confirmation.
- `assets/`: Fiat Fitness Project images used by the published page.
