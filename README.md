# Healthy Fitness and Wellness

Static English website inspired by faith-friendly fitness program sites. It includes a centered brand masthead, program options, a weekly focus selector, welcome section, testimonials, and contact form.

## GitHub Pages

This project is published from the `main` branch using GitHub Pages from the repository root.

## Files

- `index.html`: main site structure.
- `styles.css`: visual and responsive styles.
- `script.js`: interactive weekly-goal selector and contact-form confirmation.
